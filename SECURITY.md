# Reporting Security Issues

The ZeroMount team takes security issues seriously. We appreciate responsible disclosure and will acknowledge your contributions.

To report a security issue, use the GitHub Security Advisory ["Report a Vulnerability"](https://github.com/Enginex0/zeromount/security/advisories/new) tab, or reach out directly in the [Telegram group](https://t.me/superpowers9).

We will respond with next steps and keep you informed of progress towards a fix.
